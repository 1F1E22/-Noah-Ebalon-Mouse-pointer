# -Noah-Ebalon-Mouse-pointer
诺亚伊贝伦鼠标指针
解压缩后打开NoahMouse文件夹。
右键点击【右键安装.inf】文件，选择【安装】。
在【方案】下拉菜单里，可以看到NoahMouse选项。选择后点击下方【确定】按钮即可使用本指针。

如果途中有报错不一定是安装失败。
把所有弹出的窗口点击确认关闭后，检查【方案】下拉菜单。
如果有标题为NoahMouse的条目出现即为安装成功；若没有，请重新尝试安装步骤。一般至多2次就能成功。
